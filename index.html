OCR Platform : Arabic Language

OCR Platform : Arabic Language

This project was undertaken during my internship at IIT Indore, in collaboration with the Arabic government. The primary objective was to develop an Optical Character Recognition (OCR) system tailored specifically for Arabic, a language that presents unique technical challenges due to its right-to-left writing orientation and context-dependent character shapes.

Unlike English OCR systems, which are widely available and benefit from standardized left-to-right patterns, Arabic requires a more nuanced approach. Characters in Arabic can take different forms depending on their position in a word, and the overall meaning can shift based on context. Existing OCR platforms struggle with these intricacies, highlighting the need for a custom-built solution.

To address these challenges, I designed and trained a robust OCR model using deep learning techniques—particularly Convolutional Neural Networks (CNNs)—for character recognition. I integrated Natural Language Processing (NLP) methods to enhance context awareness, improving the system’s ability to interpret complex document layouts and handwritten scripts accurately.

The final model achieved an impressive 96% confidence score, demonstrating strong performance in both printed and scanned Arabic texts. To ensure accessibility and usability, I developed a fully responsive, front-end interface using HTML, CSS, JavaScript, and Bootstrap. This interface allows users to upload documents, view extracted text, and interact with the model in real time.

Overall, this project not only improved my understanding of OCR systems for non-Latin scripts but also allowed me to deliver a practical solution to a real-world linguistic challenge.

Natural Language to SQL Bot

Natural Language to SQL Bot

This self-initiated project aims to simplify database querying for non-technical users by translating natural language inputs into structured SQL queries. The tool demonstrates the power of combining Large Language Models (LLMs) with NER (Named Entity Recognition) and RAG (Retrieval-Augmented Generation) techniques to deliver context-aware, accurate query generation.

At its core, the system leverages the Google Gemini API (via google.generativeai) to process and interpret user prompts. It is enhanced with spaCy's NER capabilities to detect relevant entities—such as dates, product names, or cities—within user queries. These entities help ground the prompt with real-world context and align it with the database schema.

To further improve precision, the project incorporates a RAG-style example injection mechanism, where relevant few-shot examples are dynamically selected and added to the prompt. This helps guide the LLM toward generating syntactically correct and semantically meaningful SQL statements.

The application supports a MySQL-style schema with three interrelated tables—customers, orders, and order_items—and uses schema grounding to inform the LLM of field names and table relationships. These schema details are dynamically loaded from a separate schema.py module and passed into the prompt to enhance relevance and reduce ambiguity.

The project is deployed via a clean and minimal Streamlit interface, allowing users to interact easily, input questions, and view the corresponding SQL output in real time. This UI-centric approach ensures accessibility for both technical and non-technical users.

By combining prompt engineering, structured schema injection, and modular NLP tools, this project showcases an effective approach to automated query generation—a crucial component in modern data access, enterprise analytics, and AI-assisted BI tools.

Quora Question Pairs

Quora Question Pairs : Identifying Duplicate Questions

This self-initiated Natural Language Processing (NLP) project aimed to build a model that could automatically identify whether two Quora questions are semantically similar or duplicates. The goal was to help platforms like Quora minimize redundancy and improve user experience by flagging or merging similar content.

The project involved building a supervised classification model that achieved an accuracy of 80.65% in identifying duplicate question pairs. To do this, extensive text preprocessing was performed, including the removal of HTML tags, punctuation, stopwords, and special characters, followed by lemmatization to normalize the input data.

Beyond standard preprocessing, the feature engineering phase played a key role. Custom features were crafted to capture semantic relationships between question pairs—such as common word counts, total word counts, token-based overlap metrics, and TF-IDF weighted cosine similarities. These features provided the model with deeper insights into the context and intent of each question.

For modeling, both Random Forest and XGBoost classifiers were employed. Hyperparameter tuning and cross-validation were conducted to ensure robustness and prevent overfitting. The combination of advanced feature engineering and ensemble-based learning models helped deliver high-quality predictions on unseen data.

This project showcases a blend of NLP pipeline construction, feature engineering, and machine learning model design, simulating real-world scenarios where platforms need to manage redundant user-generated content effectively.

Prediction of Amazon Cart

Prediction of Amazon Cart Prices : A Multivariate Parameter Approach

This self-initiated project focused on predicting the total price of Amazon shopping carts using multivariate regression techniques. The goal was to accurately estimate cart value based on various product and user features, simulating real-world e-commerce forecasting challenges.

A comprehensive machine learning pipeline was developed to handle the entire workflow—from data cleaning and feature engineering to model training and evaluation. The pipeline incorporated transformers for handling missing values, encoding categorical variables, and scaling numerical features.

To improve model performance, RandomizedSearchCV was used for hyperparameter tuning across different regression models, including Linear Regression, Random Forest, and XGBoost Regressor. XGBoost and Linear Regression were particularly effective in handling null values and optimizing the preprocessing steps.

Advanced feature engineering was performed to extract meaningful insights from the data, such as total item quantities, average price per product, and cart size. These features significantly contributed to model accuracy.

The final model, optimized with a Random Forest Regressor, achieved a high R² score of 0.94, indicating excellent predictive power. This project highlights your ability to combine data preprocessing, regression modeling, and hyperparameter optimization to solve practical business problems in e-commerce analytics.

Architecture: Thesis

Family Oriented Orphanage

Nurturing Homes : Family Oriented Orphanage with Education and Traininng

This architectural thesis envisions a compassionate, non-institutional living environment for orphaned children. Set on a 26,500 sq.m. site, the design features 18+ individual dwelling units—each housing six children and one dedicated woman caretaker, such as a widow or a socially excluded woman—fostering a sense of family and emotional support.

The campus is holistically planned with an administrative block, school, vocational training center, playground, landscaped park, director's residence, staff housing, and open-air seating areas to encourage social interaction and learning. The project emphasizes inclusivity, self-sufficiency, and emotional wellbeing, inspired by successful case studies like SOS Children’s Village Aqaba and Falatow Jigiyaso Orphanage.

By integrating education, training, and dignified care within a family-oriented setting, the project challenges conventional orphanage models and aims to empower children for independent adulthood while addressing loneliness, social stigma, and reintegration.

Senior Housing Project

Solaceia : Senior Housing Community Project

This architectural studio project envisioned a thoughtfully designed senior housing community to accommodate approximately 200 elderly individuals, integrating comfort, independence, and well-being within a resort-like setting. The site, characterized by natural contours, provided an opportunity to design a landscape-sensitive solution that embraced the terrain while ensuring full accessibility.

The residential units were a combination of 1BHK, 2BHK, 3BHK houses, and studio apartments, offering a variety of options to cater to the diverse needs of the aging population. Each unit was designed with user-centric principles—ensuring natural light, ventilation, and ease of mobility—supporting both independent and semi-assisted living.

The master plan incorporated key communal amenities such as a central community center for social interaction, an amenity block for wellness and recreational activities, and a dedicated restaurant providing elderly-friendly meal options. Additional provisions included a swimming pool, administrative office, well-planned parking, and accessible circulation throughout the site via ramps and walkways.

This project aimed to foster a sense of belonging, dignity, and purpose for senior residents, blending architectural functionality with a resort-style ambiance to support healthy, active aging within a self-sustained and inclusive environment.
